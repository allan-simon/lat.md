# Docs

See [[src/app.php#greet]] for the greeting function.

The [[src/app.php#Greeter]] class has a method [[src/app.php#Greeter#greet]] and a constant [[src/app.php#Greeter#DEFAULT_NAME]].

The [[src/app.php#Greeting]] interface and the [[src/app.php#Loggable]] trait (with [[src/app.php#Loggable#log]]) define behaviour.

The [[src/app.php#Color]] enum has case [[src/app.php#GREEN]] (also [[src/app.php#Color#GREEN]]) and method [[src/app.php#Color#label]].

Default greeting is [[src/app.php#DEFAULT_GREETING]].
